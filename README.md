# Titan Attire Frontend

A modern and responsive e-commerce frontend built using React. This project provides a clean shopping experience with product browsing, authentication pages, and a user-friendly interface.

## Features

* Responsive UI
* Product Listing
* Product Details Page
* Shopping Cart
* Login & Signup Pages
* Modern React Components

## Tech Stack

* React
* JavaScript
* CSS
* React Router

## Installation

bash
git clone https://github.com/Sanila-rashid/titan-attire-frontend.git
cd titan-attire-frontend
npm install
npm start
